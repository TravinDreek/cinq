# CINQ
CINQ (**C**INQ **I**s **N**ot **Q**Q) 是一个自由的 QQ 客户端. 这是一个实验性项目 (WIP).

## 环境需求:
- [PHP 7.4 Thread Safe](https://www.php.net/downloads)
- [parallel](https://www.php.net/manual/en/parallel.setup.php)
- [go-cqhttp](https://github.com/Mrs4s/go-cqhttp)
